# Fuentes

Esta carpeta contiene los archivos de fuente **Fendesert Sans**.

## Archivos necesarios

Para que la fuente funcione correctamente, agregá los siguientes archivos en esta carpeta:

- `FendesertSans-Regular.woff2` (recomendado)
- `FendesertSans-Regular.woff` (fallback)
- `FendesertSans-Regular.ttf` (fallback adicional)
- `FendesertSans-Bold.woff2` (recomendado)
- `FendesertSans-Bold.woff` (fallback)
- `FendesertSans-Bold.ttf` (fallback adicional)

## Nota

Si no tenés los archivos de la fuente, la aplicación usará **Oswald** como fuente de respaldo para los títulos.
